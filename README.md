# mono-vio
this is a monocular vio solution for slam system

# Dependencies and Requirements
## Dependencies:
+ Qt5Core
+ Qt5Gui
+ Qt5Widgets
+ Qt5Charts
+ G2O
+ Sophus
+ OpenCV
+ yaml-cpp
+ Eigen3
+ Doxygen (optional)
## Requirements:
+ cpp 17 or above (Mandatory)
+ cmake 3.22.x or above (Optional)
+ vcpkg (Optional)
+ Visual Studio 2022 (Optional,if you are using windows)
